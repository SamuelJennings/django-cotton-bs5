Dropdowns display contextual menus. Use the text attribute for the button label and variant for button color. Inside, use c-dropdown.item for menu items, c-dropdown.header for section headers, and c-dropdown.divider for separators. {% cotton:verbatim %} {% endcotton:verbatim %} Use the direction attribute to control where the menu opens. Values: "down" (default), "up", "start", "end". {% cotton:verbatim %} {% endcotton:verbatim %}
