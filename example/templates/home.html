Cotton Bootstrap 5

Beautiful, reusable Bootstrap 5 components for Django Cotton. Build modern web interfaces faster with clean, semantic template syntax.

GitHub PyPI

Quick Start

1. Install the package
pip install django-cotton-bs5
2. Add to your INSTALLED_APPS in settings.py 
INSTALLED_APPS = [
    # ... your other apps
    "django_cotton",
    "cotton_bs5",
]
That's it! You can now use Cotton BS5 components in your Django templates.
Reduce Boilerplate

Cut your HTML code by ~50% or more. Replace verbose Bootstrap markup with clean, intuitive Cotton component syntax.

Easy Updates

Update component appearance site-wide by modifying a single template. Override components to match your design system effortlessly.

Consistent Styling

Maintain a uniform look and feel across your entire site. Define components once, use everywhere for guaranteed consistency.
